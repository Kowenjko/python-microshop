# FastAPI Microshop

A simple microservice shop application built with FastAPI.

## Features

- Fast and high performance
- Easy to use with automatic API documentation
- Built-in data validation
- Modern Python 3.6+ with type hints
- Asynchronous support

## Requirements

- Python 3.6+
- FastAPI
- Uvicorn

## Installation

```bash
uv add fastapi
```

## Quick Start

Run the application:

```bash
uv run fastapi dev main.py
```

Visit `http://localhost:8000/docs` for interactive API documentation.
